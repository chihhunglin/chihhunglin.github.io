Showterm
1. 免費
2. 方便分享

# 步驟
1. 安裝Ruby
```sh
sudo apt-get install ruby rubygems
```

2. 安裝showterm
```sh
sudo gem install showterm
```

3. 使用 Showterm
```sh
showterm
...
exit
```
