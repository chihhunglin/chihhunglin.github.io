使用 Pelican 創造自己的 Github Pages blog

Pelican 用 python 寫成的靜態網頁產生工具．

設定 github.io 網頁

  • 建立兩個容器
    • 取得 GitHub 帳號
    • 建立 repository ，命名規則 username.github.io-src and username.github.io(初始化加入README)

安裝 Pelican

pip install pelican markdown
git clone https://github.com/username/username.github.io-src ghpages
cd ghpages

如何擁有自己的 GitHub 網站

步驟

  1. 建立容器
    • 取得GitHub帳號
    • 建立repository，命名規則 username.github.io
  2. 複製容器
git clone https://github.com/username/username.github.io
  1. 世界您好
cd username.github.io
echo "Hello World" > index.html
  1. 上傳
git add --all
git commit -m "Initial commit"
git push

記得輸入帳號密碼
