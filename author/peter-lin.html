  2. 如何擁有自己的 GitHub 網站

    二 09 12月 2014
    By Peter Lin

    步驟

    1. 建立容器
      • 取得GitHub帳號
      • 建立repository，命名規則 username.github.io
    2. 複製容器
    git clone https://github.com/username/username.github.io
    1. 世界您好
    cd username.github.io
echo "Hello World" > index.html
    1. 上傳
    git add --all
git commit -m "Initial commit"
git push

    記得輸入帳號密碼
